# Loan-Calculator

A simple loan calculator for a class assignment. Practices layout constraints. The app will size correctly for the following iPhone displays in both portrait and landscape.

 - iPhone 13 Pro Max
 - iPhone 11 Pro Max
 - iPhone 11
 - iPhone 13 Pro
 - iPhone 13 Mini
 - iPhone 8 Plus
 - iPhone SE (3rd Gen)
